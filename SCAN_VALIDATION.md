# Scan validation checklist (do this before exam day)

Use the **same printer and paper** teachers will use on exam day.

## Setup

- [ ] Answer key entered in the app for a test subject
- [ ] 20 blank answer sheets printed at **100% scale** (Actual size)
- [ ] Fill 10 sheets with a known pattern (e.g. all A, mixed, one double-mark)
- [ ] Leave 10 sheets blank for edge-case tests

## Preview vs capture

The in-app preview should **fill the green corner frame** edge-to-edge (like a normal camera viewfinder):

- Hold phone **~30–40 cm** above a flat sheet with bright, even light
- Tap the **paper** on screen → wait a moment → capture
- Pass if preview fills the frame **and** **≥ 95%** of filled sheets score correctly

If the in-app camera will not start at all, use **Advanced → Try system camera** in scanner settings (troubleshooting only).

## Score check

For each filled sheet, compare:

- App score vs your manual count from the answer key
- Target: **≥ 95%** sheets correct without manual fix

## If scans fail

- Re-print at 100% scale
- Increase print darkness / replace toner
- Scan with more light and hold phone steadier
- Use **Review** for any low-confidence sheet — never override blindly
